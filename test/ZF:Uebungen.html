    • Manche Tags besitzen Attribute

  • id: dient als Indentifkikation fuer Elemente
  • class: CSS-Klasse im Stylesheet
  • style: Eingebettetes CSS­Stylesheet­Liste
  • title: Erleuterung zum Element
  • data-*: Spezielle Daten fuer das Element
  • contenteditable: Editierbarkeit des Elementinhalts (true/false)
  • draggable / dropzone: Drag und Drop
  • contextmenu: Context Menu (Rechtsklick)(Wird nur von Firefox unterstützt
  • accesskey: Shortcut Taste fuer Selektion
  • tabindex: Tab Reihenfolge
  • translate-*: yes/no fuer Uebersetzung (Wird nicht mehr Unterstützt)
